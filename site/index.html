How to read this. Platforms are listed alphabetically — no personal ranking; click any row to expand its full detail, and use the tiles to filter. ★ stars next to each name are current GitHub stargazers (a rough popularity signal); closed-source products show — ☆. The Bottom line is a neutral summary of real user sentiment (from Reddit, Hacker News, GitHub & reviews — praise + main gripe), refreshed periodically; very new tools are flagged as having little feedback yet. “Multiple logins” = isolate separate Claude/agent subscriptions per project. Self-hosted ✓ = runs entirely on your own machines (not the vendor's servers) — free-but-cloud-only scores ➖/~. Mobile: ✓ dedicated app or mobile-first · ~ usable from a mobile browser, no native app · – desktop / SSH only.

Capabilities from official docs & repos checked June 20, 2026; GitHub stars & user sentiment sampled the same week (both move — treat as a snapshot). “–”/“~” used instead of guessing. Projects = the platform's own project/task board or PM-tool integration (Linear, Jira, GitHub Projects). Prices are list-price estimates — verify before buying.
